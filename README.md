# RAPPY
<code>RAPPY (ラッピー) | HTML トランスコンパイラ</code>  
より迅速 (rapid) に HTML を書きましょう !

## Why RAPPY ?
純粋な HTML には無駄が多いです。  
しかし、RAPPY はどうでしょう。差は圧倒的です。
### doctype 宣言
- 純粋な HTML
```html
<!doctype html>
<html lang="ja">
```
- RAPPY
```
! ja
```

### class を付与 した hedding
- 純粋な HTML
```html
<h1 class="myClass">HTML</h1>
```
- RAPPY
```
.h1 myClass HTML
```
